2.Tutorial: Netzwerkkommunikation I
Dieses Tutorial besteht aus den Dateien countdown_package.py, countdown_server.py
und countdown_client.py
Hier wird eine grundlegende Art erklärt wie Netzwerkkommunikation über TCP mit
Python möglich ist, indem das Programm aus dem vorherigen Beispiel nun ins
Netzwerk gebracht wird.
countdown_package.py enhält dafür nurnoch die Funktionen für die Countdowns,
aber keinerlei Aufrufe derselbigen mehr.
countdown_client erlaubt es einem User die bekannten Befehle an einen Server
seiner Wahl zu schicken.
countdown_server ruft dann bei sich die Countdowns auf und zeigt sie bei sich
an.
Um dieses Beispiel auszuführen sollten sowohl der Client als auch der Server
auf mindestens einem unterschiedlichen Terminal laufen. Habt ihr beide
Terminals auf dem gleichen Rechner laufen gebt als IP-Adresse einfach localhost
an. Ich habe das Programm leider nicht mit mehreren Rechnern testen können, da
ich gerade nur meinen Laptop zur Verfügung habe, aber unter Angabe der
entsprechenden IP-Adressen sollte auch das ohne Probleme funktionieren (Achtet
auf Meldungen eurer Firewall, solltet ihr das probieren und Fehler bekommen)
